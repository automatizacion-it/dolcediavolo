# ☕ Dolce Diavolo — Caffè Colombiano

> «Negro como el diablo, caliente como el infierno, puro como un ángel, dulce como el amor.»
> — Charles-Maurice de Talleyrand

Sitio web oficial de la marca **Dolce Diavolo**: café colombiano de origen con espíritu italiano.

🌐 **Producción:** [dolcediavolo.com](https://dolcediavolo.com)

## Páginas

| Página | Archivo | Descripción |
|---|---|---|
| Inicio | `index.html` | Viaje scroll por los 4 versos de Talleyrand (Nero → Caldo → Puro → Dolce) |
| Tienda | `tienda.html` | E-commerce: 3 tostados (Inferno, Purgatorio, Paradiso) + accesorios |
| Origen | `origen.html` | Historia de la finca, proceso del grano a la taza |
| Rituali | `preparaciones.html` | 7 métodos de preparación, del espresso al cold brew |
| Etiquetas | `etiquetas.html` | Hoja de diseño de empaque (interna, no enlazada en nav) |

## Stack

- HTML/CSS/JS vanilla, sin dependencias ni build
- Google Fonts: Cinzel, Cormorant Garamond, Jost (licencia OFL)
- SVG inline para logo y mockups de producto
- IntersectionObserver para fases de color y revelado por scroll
- GitHub Pages + dominio Cloudflare
